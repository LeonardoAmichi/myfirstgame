# Rei da Pradaria - AppGameKit

Um jogo de tiro frenético e desafiador desenvolvido em **BASIC** utilizando a engine **AppGameKit**. Este projeto foi fortemente inspirado no mini-game "O Rei da Pradaria" (*Journey of the Prairie King*) do aclamado **Stardew Valley**.

## Sobre o Projeto
O objetivo principal foi recriar a mecânica de sobrevivência e tiro multidirecional, onde o jogador assume o papel de um cowboy que deve enfrentar hordas crescentes de orcs vindos de todas as direções. O projeto foca em mecânicas sólidas de colisão, sistema de power-ups e progressão de dificuldade.

## Funcionalidades
- **Movimentação Clássica:** Controle total do personagem usando as teclas `W`, `A`, `S`, `D`.
- **Sistema de Tiro:** Mecânica de tiro direcionada pelo mouse/ponteiro.
- **Inimigos Dinâmicos:** Orcs que perseguem o jogador com velocidade progressiva conforme o tempo passa.
- **Power-ups & Itens:**
  - **Botas:** Aumentam a velocidade de movimento do herói.
  - **Arma Melhorada:** Aumenta a velocidade dos projéteis.
  - **Vidas Extra:** Corações que surgem no mapa para recuperar sua saúde.
- **Trilha Sonora Integrada:** Músicas de fundo para menu e gameplay, além de efeitos sonoros para tiros, mortes e power-ups.
- **Interface Completa:** Menus de início, ajuda e tela de Game Over com exibição de pontuação (Kills e Tempo).

## Tecnologias Utilizadas
- **Linguagem:** BASIC (AppGameKit Script)
- **Engine:** [AppGameKit Classic/Studio](https://www.appgamekit.com/)
- **Assets:** Sprites 2D animados e trilha sonora imersiva.

## Como Rodar
1. Certifique-se de ter o **AppGameKit Classic** ou **Studio** instalado.
2. Clone este repositório.
3. Abra o arquivo `MyFirstGame.agk` na engine.
4. Clique em **Run** para compilar e jogar.
